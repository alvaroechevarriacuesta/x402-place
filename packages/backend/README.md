# @x402place/backend

Backend API server for x402place.

## Dependencies

- `@x402place/shared` - Shared utilities, types, and database client

## Development

```bash
pnpm dev
```

## Structure

- `src/server.ts` - Server entry point
- `src/routes/` - API route handlers
- `src/ws.ts` - WebSocket handling
